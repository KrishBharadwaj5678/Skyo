# 🌤️ Skyo

**Skyo** это современное, чистое и адаптивное погодное приложение, которое предоставляет актуальную информацию о погоде и качестве воздуха для любого города в мире.

![Skyo Preview](https://github.com/KrishBharadwaj5678/Skyo/raw/main/SkyoDemo.png)

## 🚀 Возможности

| Возможность                  | Описание                                            |
| ---------------------------- | --------------------------------------------------- |
| 🌍 Поиск города              | Поиск погодных данных для любого города в мире      |
| 🌡️ Текущая температура      | Отображение актуальной температуры в °C             |
| 🌬️ Данные о ветре           | Скорость и направление ветра                        |
| 💧 Влажность                 | Процент влажности в реальном времени                |
| 👁️ Видимость                | Дальность видимости в километрах                    |
| 🌅 Восход и закат            | Точное время восхода и заката солнца                |
| 🌡️ Ощущаемая температура    | Температура, воспринимаемая человеком               |
| 🧪 Данные о качестве воздуха | Информация о загрязнении воздуха в реальном времени |
| 🧫 Уровни загрязнителей      | PM2.5, PM10, NO₂, NO, CO, SO₂, NH₃, O₃              |
| 🌙 Тёмная / светлая тема     | Удобное переключение темы                           |
| 📱 Адаптивный дизайн         | Оптимизация для мобильных устройств, планшетов и ПК |
| ⚡ Быстрый интерфейс          | Плавные переходы и мгновенные обновления            |

## 🔐 Переменные окружения

Создайте файл `.env` в корне проекта и добавьте:

```env
VITE_BASE_URL = "https://api.openweathermap.org/data/2.5"
VITE_APP_ID = your_api_key_here
```
